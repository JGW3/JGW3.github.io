RemindMi

This is the result of my group's Software Engineering semester project for Fall 2022.
RemindMi is a to-do list and reminder app designed with an emphasis on users with special needs. The primary feature of the app is an admin account that can create, edit, and delete tasks, while dependent accounts can only "complete" their assigned tasks.
RemindMi is a mobile that can run on Android or iOS.
